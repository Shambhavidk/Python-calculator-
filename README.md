# Python Calculator

## 📌 Description
This is a simple command-line calculator built using Python.
It performs basic arithmetic operations:
- Addition
- Subtraction
- Multiplication
- Division

## 🛠 Technologies Used
- Python

## ▶ How to Run
1. Install Python on your system
2. Download the calculator.py file
3. Run using:
   python calculator.py
